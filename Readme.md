# FMS Project

## Overview

FMS (File Management System) is a web-based application designed to streamline file management tasks. The project consists of a frontend and backend, each organized in their respective directories.

---

## Project Structure

```
FMS/
├── frontend/
│   └── ... (React app, UI components, assets)
├── backend/
│   └── ... (API, database models, controllers)
└── README.md
```

---

## Frontend

- **Tech Stack:** React, JavaScript, CSS
- **Features:**
    - User authentication (login/register)
    - Dashboard for managing files
    - Upload, download, and delete files
    - Responsive design for desktop and mobile
- **Setup:**
    1. Navigate to `frontend/`
    2. Run `npm install`
    3. Start with `npm start`

---

## Backend

- **Tech Stack:** Node.js, Express, MongoDB
- **Features:**
    - RESTful API for file operations
    - User authentication and authorization
    - File storage and retrieval
    - Error handling and validation
- **Setup:**
    1. Navigate to `backend/`
    2. Run `npm install`
    3. Start with `npm run dev`

---

## Getting Started

1. Clone the repository.
2. Install dependencies for both frontend and backend.
3. Configure environment variables as needed (see `.env.example`).
4. Start backend and frontend servers.
